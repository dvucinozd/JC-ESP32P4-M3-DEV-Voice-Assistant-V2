=================================================================
ESP32-P4 Voice Assistant - Build & Flash Instructions
=================================================================

Due to ESP-IDF requirements, build must be run in Windows Command Prompt
or ESP-IDF PowerShell (not Git Bash).

-----------------------------------------------------------------
METHOD 1: Using Build Scripts (RECOMMENDED)
-----------------------------------------------------------------

1. Open Windows Command Prompt (cmd.exe)

2. Navigate to project:
   cd D:\AI\ESP32P4\JC-ESP32P4-M3-DEV-Voice-Assistant_NEW

3. Run build:
   build.bat

4. Run flash (after successful build):
   flash.bat

-----------------------------------------------------------------
METHOD 2: Manual Commands
-----------------------------------------------------------------

1. Open ESP-IDF Command Prompt:
   - Start Menu → ESP-IDF → ESP-IDF 5.5 CMD

2. Navigate to project:
   cd D:\AI\ESP32P4\JC-ESP32P4-M3-DEV-Voice-Assistant_NEW

3. Build:
   idf.py build

4. Flash to board (COM13):
   idf.py -p COM13 flash monitor

   To exit monitor: Ctrl+]

-----------------------------------------------------------------
METHOD 3: Using VS Code
-----------------------------------------------------------------

1. Open folder in VS Code:
   D:\AI\ESP32P4\JC-ESP32P4-M3-DEV-Voice-Assistant_NEW

2. Install ESP-IDF extension if not already installed

3. Use Command Palette (Ctrl+Shift+P):
   - ESP-IDF: Build project
   - ESP-IDF: Flash device
   - ESP-IDF: Monitor device

-----------------------------------------------------------------
EXPECTED BUILD TIME
-----------------------------------------------------------------

First build: 5-10 minutes (compiles ESP-IDF framework)
Subsequent builds: 30-60 seconds (only changed files)

-----------------------------------------------------------------
TROUBLESHOOTING
-----------------------------------------------------------------

Build fails with "component not found":
  → Check common_components folder exists
  → Run: dir common_components

No sound from speaker:
  → Check speaker connection
  → Check PA_EN pin (GPIO11)
  → Verify ES8311 codec initialization in logs

Board not detected:
  → Check Device Manager for COM13
  → Try different USB cable
  → Press Reset button
